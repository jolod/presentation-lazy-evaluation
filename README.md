# Lazy Evaluation

By Johan Lodin, presented 2019-10-30 at [Got.λ](https://www.meetup.com/got-lambda/).

## Building the presentation

To produce `slideshow.html`, run

```bash
npm run once
```

To automatically update `slideshow.html` when `slides.md` changes, run

```bash
npm run watch
```

## System dependencies

* npm
